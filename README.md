# 🎨 Meu Portfólio Pessoal

![Status do Projeto](https://img.shields.io/badge/Status-Em%20Andamento-ff69b4?style=for-the-badge)

Bem-vindo ao repositório do meu portfólio! Este projeto está sendo desenvolvido com **HTML5, CSS3 e JavaScript puros** para demonstrar meus fundamentos em desenvolvimento Front-End e UI/UX Design.

## 🚀 O que estou construindo?
O objetivo deste espaço é unir minha base acadêmica em Design de Animação e Jogos Digitais com a performance do código web, criando uma interface fluida, responsiva e com a minha identidade visual.

## 🛠️ Tecnologias em uso
* **HTML5** (Semântica e acessibilidade)
* **CSS3** (Layouts com Flexbox/Grid e estilização personalizada)
* **JavaScript** (Manipulação de DOM e interatividade)

> **Nota:** Este projeto está em constante evolução! Atualmente, estou refinando a responsividade e a seção de projetos. ✨
